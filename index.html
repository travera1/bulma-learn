Bulma?

Bulma is a modern CSS framework based on Flexbox. Using Bulma, we can describe our website's design using just classes.

...Flexbox?

Flexbox is a CSS spec that makes sectining and aligning more natural. We don't need to know Flexbox but it's how Bulma works behind-the-scenes.

...Classes?

Instead of writing our CSS per-element, we can use predefined classes. With enough classes, we can descirb our website using semantics.
